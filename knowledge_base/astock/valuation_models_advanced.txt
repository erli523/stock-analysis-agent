# 股票高级估值模型详解

## 一、相对估值法

### 1.1 市盈率（P/E）进阶分析

**PE计算变种**
- **静态PE**：当前股价 / 上年度每股净利润（滞后性强）
- **动态PE（TTM PE）**：当前股价 / 过去12个月每股净利润（最常用）
- **前瞻PE（Forward PE）**：当前股价 / 未来12个月预测每股净利润（基于分析师预测）

**PE适用场景**
- 适合：盈利稳定、持续正收益的成熟企业
- 不适合：亏损企业（PE为负无意义）、强周期企业（PE在顶峰时最低、底部最高，产生"PE陷阱"）

**周期性行业PE陷阱**
以钢铁、煤炭为例：
- 行业顶峰（高价格）：盈利最高 → PE看起来最低 → 市场被低PE吸引大量买入
- 行业底部（低价格）：亏损或微利 → PE极高或为负 → 市场因PE高恐慌卖出
- 正确方法：周期股应用PB估值或中枢盈利PE估值

### 1.2 市净率（P/B）

**公式**：P/B = 当前股价 / 每股净资产

**适用场景**
- 金融行业（银行、保险）：资产负债表结构清晰，PB估值为核心
- 重资产行业（钢铁、地产）：资产价值相对稳定
- 破净股（PB<1）：通常意味着市场认为该公司资产面临减值风险，或长期ROE低于资本成本

**PB-ROE框架**（戈登增长模型派生）
合理PB = ROE / (Ke - g)
其中 Ke 为权益成本，g 为长期增长率
- ROE越高 → 合理PB越高（高ROE公司值得溢价）
- 当前PB远低于ROE所隐含的合理PB → 低估

### 1.3 PEG估值

**公式**：PEG = PE / 盈利增速（%）

**解读**
- PEG < 1：可能低估（成长未完全定价）
- PEG = 1：合理估值
- PEG > 2：可能高估（高增长预期已充分反映）

**PEG适用场景**
- 成长型公司（增速>15%的科技、医药、消费成长）
- 不适用于：增速<10%的成熟行业、亏损公司、周期性行业

**PEG计算注意事项**
- 增速应取未来3年CAGR（复合年增长率），而非单年预测（单年波动大）
- 增速的可信度至关重要：分析师预测增速的偏差是PEG失效的主因

## 二、EV/EBITDA估值

### 2.1 基本概念

**EV（企业价值）**
EV = 市值 + 有息负债 - 现金及现金等价物
= 收购该公司需支付的真实价格

**EBITDA（息税折旧摊销前利润）**
EBITDA = 净利润 + 所得税 + 利息费用 + 折旧 + 摊销
= 公司的经营性现金创造能力

**EV/EBITDA核心优势**
1. 排除了资本结构差异（不同杠杆率公司可比）
2. 排除了税率差异（跨国比较有效）
3. 排除了折旧政策差异（资本密集型行业尤其有用）

### 2.2 各行业EV/EBITDA参考范围

| 行业 | EV/EBITDA合理范围 | 说明 |
|------|-----------------|------|
| 科技（互联网） | 15-30x | 轻资产，折旧少 |
| 消费白酒 | 20-40x | 高现金流，强品牌 |
| 医药制造 | 12-20x | 中等资产密集 |
| 制造业（汽车） | 8-12x | 重资产，周期性 |
| 公用事业 | 8-15x | 稳定现金流 |
| 银行/金融 | 不适用 | 财务特殊，EBITDA概念模糊 |

### 2.3 EV/EBITDA的局限性

- 高度依赖债务结构：高负债公司EV大，会人为抬高比值
- 忽略资本开支（CAPEX）：对重资产公司，折旧后还需大量再投资
- 因此常结合EV/EBIT（不加回折旧，更接近真实盈利）使用

## 三、DCF（折现现金流）模型

### 3.1 模型架构

DCF是将公司未来所有自由现金流按折现率折算为现在价值的绝对估值方法。

**股权自由现金流（FCFE）**
FCFE = 净利润 + 折旧摊销 - 资本开支 - 营运资本变动 + 净借款

**加权平均资本成本（WACC）**
WACC = Ke × (E/V) + Kd × (1-t) × (D/V)
- Ke = 权益成本（CAPM模型：Ke = 无风险利率 + β × 市场风险溢价）
- Kd = 债务成本
- E/V = 权益占比，D/V = 负债占比

**两阶段DCF模型**
```
内在价值 = Σ(FCFE_t / (1+WACC)^t) + 终值 / (1+WACC)^N

终值 = FCFE_(N+1) / (WACC - g)
其中 g 为永续增长率（通常取1-3%，不超过GDP增速）
```

### 3.2 DCF关键假设的敏感性

**折现率（WACC）的影响**
| WACC变化 | 对估值影响 |
|---------|----------|
| WACC从10%降到8% | 内在价值可能上升20-30% |
| WACC从8%升到12% | 内在价值可能下降25-35% |

**永续增长率（g）的影响**
- g 从2%升到3%：对终值影响巨大（终值占总估值通常超过60%）
- 因此DCF对输入假设极其敏感，被戏称为"垃圾进垃圾出（GIGO）"

**实践中DCF的使用方式**
- 不单独用DCF定价，而是用DCF做区间估值（牛/熊/基准三种假设）
- 用于验证相对估值（PE/PB）得出的结论是否合理
- 反向DCF：已知当前市价，反推市场对该公司增速/利润的隐含假设，判断是否合理

### 3.3 A股DCF模型参数参考

- 无风险利率：当前10年期国债收益率（约2.0-2.5%）
- 市场风险溢价（ERP）：A股历史数据约6-7%
- β值：成熟消费品约0.8，科技成长约1.3-1.8，周期品约1.2-1.5
- 永续增长率：A股建议取1.5-2.5%（对应中国长期经济增速中枢）

## 四、股息贴现模型（DDM）

### 4.1 戈登增长模型

适用于分红稳定、增长可预期的公司：

**公式**：P = D₁ / (Ke - g)
- D₁：下一期预期股息
- Ke：权益必要报酬率
- g：股息永续增长率

**适用场景**
- 银行、公用事业、高分红传统企业（股息率>4%）
- 国有大行：分红比例稳定（35-38%），适合DDM定价

**高股息策略（红利投资）**
- 选股标准：股息率>4%、分红连续性好（10年以上）、派息率稳定
- 历史数据：高股息指数（中证红利）在震荡市中显著跑赢成长型指数
- 低利率环境下，高股息股的吸引力进一步提升（与债券的利差扩大）

## 五、市销率（P/S）与市值/GMV

### 5.1 市销率（PS）

**公式**：PS = 市值 / 营业收入

**适用场景**
- 亏损但高速成长的公司（互联网早期、新能源新赛道）
- 毛利率相对稳定的行业
- 无法用PE/PB估值的阶段（创业期）

**PS估值参考**
- 互联网平台：1-5x PS
- 软件（SaaS）：5-15x PS（高毛利+高增速）
- 工业制造：0.5-2x PS

**注意事项**
- PS忽略了盈利能力，高PS公司需要未来实现利润转化
- 当公司利润率极低（<5%）时，PS与PE比较可能失真

### 5.2 GMV倍数（电商/平台公司）

**GMV（商品交易总额）** 是电商平台经营规模指标
- 货币化率：营收/GMV，通常2-5%
- 市值/GMV倍数：合理水平通常0.1-0.5x（取决于盈利能力）

## 六、综合估值实践方法

### 6.1 多方法交叉验证

专业投资者通常不依赖单一估值方法，而是综合使用：

| 公司类型 | 主要方法 | 辅助方法 |
|---------|---------|---------|
| 消费品蓝筹 | PE（TTM/Forward）| DCF、股息率 |
| 金融银行 | PB-ROE | DDM、股息率 |
| 科技成长 | PEG、EV/Sales | DCF（反向）|
| 周期资源 | PB、EV/EBITDA | 中枢盈利PE |
| 初创亏损 | PS、EV/GMV | 同类可比公司 |

### 6.2 A股估值洼地识别

**方法1：历史分位数法**
- 当前PE/历史PE中位数分位数 < 20%：历史低估区域
- 数据来源：申万行业PE分位数（同花顺、Wind可查）

**方法2：股权风险溢价（ERP）**
- ERP = 1/PE（沪深300）- 10年国债收益率
- 历史均值约3-3.5%；ERP > 4%为低估，< 2%为高估

**方法3：PB-ROE散点图**
- 以行业内所有公司ROE为X轴，PB为Y轴
- 位于散点线以下的公司（同等ROE下PB更低）= 相对低估

### 6.3 估值调整因子

**应下调估值的因素**
- 公司治理问题（大股东占款、高管频繁减持）
- 商誉减值风险（并购溢价过高）
- 负债率过高（资产负债率>70%）
- 行业政策不确定（监管风险）

**应上调估值的因素**
- 行业龙头地位（市占率第一/第二，享有定价权）
- 持续高ROE（连续5年ROE>15%）
- 研发投入持续增加（科技型公司未来竞争力）
- 管理层持股比例高（利益与股东一致）
